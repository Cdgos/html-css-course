Etiqueta Title y Meta etiqueta description

Etiqueta title

title

Es el título del documento HTML, su contenido debe ser referente a lo que el usuario encuentre.

Regla SEO: Debe contener entre 55 y 65 caracteres.

Colocar un title diferente por cada documento html.

Meta etiqueta description

meta name="description" content="En esta página estamos aprendiendo los fundamentos de HTML con @jonmircha."

Es muy importante para el diseño responsivo de la página, siempre colocarla.

Regla SEO: Debe contener máximo 165 caracteres.

Colocar una meta description diferente por cada documento html.

name="description": Indicamos que irá la descripción de nuestro documento HTML.

content:"...": Contenido de nuestra descripción.
