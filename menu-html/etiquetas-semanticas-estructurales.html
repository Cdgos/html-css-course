Etiquetas semánticas estructurales

div - Es una etiqueta contenedora que no tiene semántica
header - Cabecera de un sitio web o de una sección.
main - Define la sección principal del documento, sólo puede exitir una etiqueta main por documento.
footer - Pié de página de un sitio web o de una sección.
nav - Representa una navegación.
article - Representa una sección de autocontenido (que por sí sola se explica). Ej: algun sitio de blogs.
aside - Representa un contenido complementario o secundario. (Al lado). Ej: Artículos relacionados ..., zonas de publicidad.
section - Representa una sección de contenido genérico.
address - Representa una dirección física.
